QUESTION

What discrepancies exist in the comptroller’s report on security guard contracting, and how does DCAS address these issues?

0:53:50

·

5 min

The comptroller’s report on security guard contracting contains inaccuracies, particularly regarding the status of contracts and negotiated rates, which DCAS is addressing.

  • The report inaccurately describes DCAS’s master contract with security guards as active when it had expired before the asylum seeker crisis.
  • It mischaracterizes task orders as active contracts and overlooks their specific scopes.
  • DCAS contends that the federal government, not DCAS, negotiates rates for security services with Garner, and agencies can negotiate down these federally set rates.
  • Agencies collaborating under DCAS, like NYSEM and H&H, are managing security services collaboratively, not at disparate rates as suggested.
  • DCAS commits to providing documentation to address perceived report inaccuracies to the committee and comptroller.
